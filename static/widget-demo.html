GammIA Widget

v1.0 · Embeddable Chatbot

Integra GammIA en tu sitio web

Un único <script> tag añade el chatbot flotante a cualquier página. Detecta automáticamente el contexto (público vs. intranet) y filtra la base de conocimiento con RBAC.

🌐 Sitio Corporativo PUBLIC

Acceso solo a documentos tagueados como public. Ideal para gammaingenieros.com.

<!-- Pega este código antes de </body> en tu sitio web --> <script src="https://gammia-api-1028680563477.us-central1.run.app/static/gammia-widget.js" data-context="public" data-theme="dark" data-position="right" ></script>
💡 Para tema claro usa data-theme="light". Para posición izquierda usa data-position="left".

🔒 Google Sites (Intranet) INTERNAL

Acceso completo a toda la base de conocimiento. En Google Sites: Insertar → Embed → Insertar código.

<!-- Para Google Sites: Insertar → Embed → Código personalizado --> <script src="https://gammia-api-1028680563477.us-central1.run.app/static/gammia-widget.js" data-context="internal" data-secret="TU_WIDGET_INTERNAL_SECRET" data-theme="dark" data-position="right" ></script>
⚠️ El data-secret debe coincidir con la variable de entorno WIDGET_INTERNAL_SECRET en Cloud Run. Sin el secret correcto, el widget degrada automáticamente a acceso público.

📋 Alternativa iFrame (Google Sites)

Si el embed de script no funciona en tu versión de Google Sites, usa este iframe.

<!-- iFrame embed — compatible con todas las versiones de Google Sites --> <iframe src="https://gammia-api-1028680563477.us-central1.run.app/widget?ctx=internal&secret=TU_SECRET" width="380" height="560" frameborder="0" style="border-radius:20px;box-shadow:0 10px 40px rgba(0,0,0,.3)" ></iframe>
